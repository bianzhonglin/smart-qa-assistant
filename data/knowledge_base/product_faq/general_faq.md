# Product FAQ

Q: What is Smart QA Assistant?
A: It is an assistant that supports both direct LLM answers and retrieval-augmented answers.

Q: When should I use retrieval mode?
A: Use retrieval mode when the answer should be grounded in your internal documents.

Q: Does retrieval mode always return sources?
A: Yes, this project prints source files used as context.

Q: Can I add more docs later?
A: Yes. Put markdown or text files under data/knowledge_base and rebuild the index.
